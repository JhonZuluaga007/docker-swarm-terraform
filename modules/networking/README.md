Provisions the VPC that the swarm cluster runs in: a single public subnet, an internet gateway, a public route table, and VPC Flow Logs shipped to CloudWatch. It also locks down the VPC's auto-created default security group so nothing can attach to it by accident.

There's no NAT gateway or private subnet here on purpose — see the "Design decisions" section in the root README for why.
